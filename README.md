# Akhilesh Kumar Portfolio

A modern and responsive personal portfolio website built using React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Features

- Modern UI Design
- Responsive Layout
- Smooth Animations
- Skills Section
- Projects Showcase
- Contact Section
- GitHub & LinkedIn Links

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

---

## 📂 Projects Included

### Snake Game
A classic Snake Game developed using Java Swing with smooth movement, score tracking, and sound effects.

### Brick Breaker Game
A Brick Breaker Game built using Java Swing with collision detection and multiple levels.

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AKHILESH-create/portfolio-akhilesh
```

Go to project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
